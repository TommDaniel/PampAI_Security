## Codebase Patterns
- phishing-api/app.py uses Pydantic BaseModel + Field for all request/response models
- API is FastAPI (not Flask), running on Uvicorn; lifespan context manager handles startup/shutdown
- DB connection uses SQLAlchemy async (create_async_engine + async_sessionmaker) with asyncpg driver
- DATABASE_URL env var: `postgresql+asyncpg://user:pass@host:5432/db` — defaults to localhost
- DB_ENABLED flag in db.py: init_db() sets True only on successful connect (graceful degradation)
- Migrations SQL in phishing-api/migrations/ — SQLAlchemy also creates tables via metadata.create_all
- Python syntax check: `python3 -c "import ast; ast.parse(open('file').read())"`
