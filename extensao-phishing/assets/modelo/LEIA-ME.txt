Coloque aqui o arquivo do modelo convertido para ONNX:

  modelo_phishing.onnx

Para converter o modelo treinado (.pkl) para ONNX, rode no Colab após o treino:

  !pip install skl2onnx

  from skl2onnx import convert_sklearn
  from skl2onnx.common.data_types import FloatTensorType
  import joblib

  model = joblib.load('gbm_phishing_brasileiro.pkl')
  onnx_model = convert_sklearn(
      model,
      initial_types=[('float_input', FloatTensorType([None, model.n_features_in_]))],
      options={'zipmap': False}   # saída de proba como Float32Array, não ZipMap
  )
  with open('modelo_phishing_ort14.onnx', 'wb') as f:
      f.write(onnx_model.SerializeToString())

O arquivo .onnx gerado deve ser colocado nesta pasta.
O número de features (model.n_features_in_) é determinado automaticamente pelo modelo — atualmente 34.
