## Codebase Patterns
- API code lives in `/workspace/phishing-api/app.py` (single-file FastAPI app)
- Model files in `/workspace/phishing-api/model/` (DomURLs-BERT, 422MB safetensors)
- Extension code in `/workspace/extensao-phishing/` using Plasmo framework (Chrome MV3)
- Training format: `[URL] <url> [WHOIS] unknown [EXTRA] key=val key=val ...` (see `benchmark-modelos/criar_dataset_multimodal.py`)
- Training used `max_length=128` (see `model/training_metadata.json`) - now correctly set in app.py
- `create_feature_text` has `whois_text` param ready for US-003 WHOIS integration
- Python environment in sandbox has no pip/packages - use AST-based structural validation for Python checks
- Node.js 20 available with npm; extension has node_modules already installed
- No network access from sandbox - cannot install packages or reach external services
